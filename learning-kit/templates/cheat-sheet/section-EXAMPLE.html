SQL — Référence Rapide

DDL — Définition

CREATE TABLE

CREATE TABLE nom (col TYPE contrainte);

Crée une nouvelle table.

CREATE VIEW

CREATE VIEW nom AS SELECT ...;

Vue classique, recalculée à chaque accès.

CREATE MATERIALIZED VIEW

CREATE MATERIALIZED VIEW nom AS SELECT ... REFRESH ON DEMAND;

Stocke physiquement le résultat.

DML — Manipulation

SELECT avec agrégat

SELECT col, COUNT(*) FROM t GROUP BY col HAVING COUNT(*) > 1;

JOIN

SELECT * FROM a INNER JOIN b ON a.id = b.id;
