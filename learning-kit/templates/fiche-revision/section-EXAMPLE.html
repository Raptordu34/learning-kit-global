Thème 1 : Fondamentaux

Qu'est-ce qu'une vue matérialisée ? Important
Une vue qui stocke physiquement son résultat, contrairement à une vue classique exécutée à chaque appel.
Quelle est la différence entre REFRESH COMPLETE et FAST ? Moyen
COMPLETE : recalcule tout. FAST : n'applique que les changements depuis le dernier refresh (nécessite un MV log).

Thème 2 : Syntaxe SQL

Syntaxe de création d'une VM Important
CREATE MATERIALIZED VIEW nom AS SELECT ... REFRESH [COMPLETE|FAST] ON [DEMAND|COMMIT];
